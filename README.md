# SDF-Lit

## Development

Setup Python venv:

```sh
python -m venv .env
source .env/bin/activate
pip install maturin
```

Build Python package after each modification:

```sh
maturin develop
```

Try it in Python:

```python
>>> import sdflit as sdf
>>> sphere = sdf.Sphere((0, 0, 0), 1)
>>> sphere.distance((0, 0, 0))
-1.0
>>> sphere.distance((1, 0, 0))
0.0
>>> sphere.distance((1, 1, 1))
0.7320507764816284
```
